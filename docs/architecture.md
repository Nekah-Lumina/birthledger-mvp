# BirthLedger System Architecture

BirthLedger uses a hybrid architecture combining blockchain verification with encrypted off-chain storage.

Components:

1. Web Application
Interface used by clinics or health workers to register maternal and child health events.

2. Smart Contracts
Blockchain programs that record birth and vaccination events.

3. Blockchain Network
A decentralized ledger that verifies and stores health event transactions.

4. Secure Data Storage
Sensitive medical data is encrypted and stored off-chain while the blockchain maintains a verification record.
